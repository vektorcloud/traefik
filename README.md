# Traefik

[![circleci][circleci]](https://circleci.com/gh/vektorcloud/traefik)

Tiny Alpine image for running [Traefik](https://traefik.io/)

## Running

    docker run --rm -ti -p 80:80 -p 8080:8080 quay.io/vektorcloud/traefik
